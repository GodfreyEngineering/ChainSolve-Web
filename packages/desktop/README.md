# @chainsolveweb/desktop

Desktop shell for ChainSolve (Tauri).

## Status

**Not started.** This directory is a placeholder for a future Tauri
integration. When work begins:

1. Run `cargo tauri init` inside this directory.
2. Set `IS_DESKTOP = true` in the shared platform flags.
3. Wire native file-system access, window management, and auto-update.

## Prerequisites

- Tauri v2+ (Rust backend, webview frontend)
- Shared engine WASM build from `crates/engine-wasm/`
