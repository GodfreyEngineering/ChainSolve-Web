# @chainsolveweb/mobile

Mobile shell for ChainSolve (Capacitor).

## Status

**Not started.** This directory is a placeholder for a future Capacitor
integration. When work begins:

1. Run `npx cap init` inside this directory.
2. Set `IS_MOBILE = true` in the shared platform flags.
3. Wire push notifications, biometric auth, and haptic feedback.

## Prerequisites

- Capacitor v6+ (web-native bridge)
- iOS: Xcode 16+, CocoaPods
- Android: Android Studio, SDK 34+
