# Support

## Getting Help

- **Bug reports**: Open a [GitHub Issue](https://github.com/GodfreyEngineering/ChainSolve-Web/issues)
  with steps to reproduce, expected vs actual behavior, and browser/OS info.
- **Feature requests**: Open a GitHub Issue with the "enhancement" label,
  or use the in-app suggestion form (Help > Submit Suggestion).
- **Security issues**: See [SECURITY.md](SECURITY.md) for responsible disclosure.
- **General questions**: Open a GitHub Discussion or contact support@chainsolve.co.uk.

## Before Filing an Issue

1. Search existing issues to avoid duplicates.
2. Include your browser, OS, and ChainSolve version (Help > About in the app).
3. If reporting a rendering or engine issue, include a screenshot or
   `.chainsolvejson` export of the affected project.
