# Archived Migrations

These iterative migrations (0001–0054) were used during initial development
to evolve the database schema incrementally.

**Do NOT run these on new Supabase projects.**

For fresh installs, use the consolidated baseline migration at:
```
supabase/migrations/0001_baseline_schema.sql
```

See [docs/DEV/SUPABASE_BOOTSTRAP.md](../../docs/DEV/SUPABASE_BOOTSTRAP.md)
for full setup instructions.

These files are kept for historical reference and to understand the
evolution of the schema over time.
